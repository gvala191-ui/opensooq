# Инструкция по использованию отправки фото

## Что изменилось

Программа теперь отправляет **одно фото** вместо нескольких текстовых паст (pasta1.txt - pasta4.txt).

## Как использовать

1. **Добавьте фото** в папку с программой (туда же, где находится main.py)
   - Поддерживаемые форматы: `.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.webp`
   - Можно добавить только одно фото - программа автоматически найдет его

2. **Запустите программу** как обычно:
   ```
   python main.py
   ```
   или
   ```
   run.bat
   ```

3. Программа автоматически найдет фото и будет отправлять его нужное количество раз каждому пользователю

## Примеры названий файлов

- `photo.jpg`
- `image.png`
- `picture.jpeg`
- `myimage.webp`

## Важно

- Файлы pasta1.txt - pasta5.txt больше **НЕ нужны** и не используются
- В папке должно быть хотя бы одно изображение
- Если фото не будет найдено, программа сообщит об ошибке и остановится
